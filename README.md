# 📝 Todo List App

A simple, interactive, and responsive To-Do List application built with pure HTML, CSS, and vanilla JavaScript. 

## ✨ Features

- **Add Tasks:** Quickly add new tasks to your list using the input field or the `Enter` key.
- **Mark as Complete:** Click on a task to toggle its completed status (strikethrough and custom checkbox).
- **Edit Tasks:** Easily modify the text of existing tasks using the edit icon.
- **Reorder Tasks:** Move tasks up and down the list using the interactive arrow icons to prioritize your work.
- **Delete Tasks:** Remove tasks you no longer need with the `x` button.
- **Data Persistence:** All tasks and their current states are automatically saved in the browser's `localStorage`. Your data remains safe even if you refresh or close the page.

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** Custom styling, UI layout, flexbox, and smooth hover animations.
- **JavaScript (Vanilla):** DOM manipulation, Event Delegation for optimized performance, and Local Storage API.

## 🚀 How to Run

No installation or server is required to run this project!
1. Clone this repository or download the source code files.
2. Ensure the `img` folder is located in the same directory as the main files.
3. Simply double-click `index.html` to open it in any modern web browser.
4. Start adding and managing your tasks!

## 📁 File Structure

- `index.html`: The main structure of the application.
- `style.css`: Contains all styling, responsive layout, and icon positioning.
- `main.js`: Contains the application's core logic and event handling.
- `/img`: A directory containing all the custom icons used in the UI (checkboxes, arrows, edit, etc.).
